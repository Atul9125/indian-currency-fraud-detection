# Satya — Fake Currency Note Verifier

FastAPI backend + custom HTML/CSS/JS frontend for the fake currency detection model (MobileNetV2 transfer learning).

## Folder structure
```
currency-app/
├── app.py                 # FastAPI server + /predict endpoint
├── fake_currency_model.h5 # your trained model
├── requirements.txt
├── templates/
│   └── index.html
└── static/
    ├── style.css
    ├── script.js
    └── uploads/            # uploaded images get saved here
```

## Run it locally

1. (Recommended) create a virtual environment:
   ```
   python -m venv venv
   venv\Scripts\activate      # Windows
   source venv/bin/activate   # Mac/Linux
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Start the server:
   ```
   uvicorn app:app --reload
   ```
   (or simply `python app.py`)

4. Open your browser at:
   ```
   http://127.0.0.1:8000
   ```

   Interactive API docs (auto-generated by FastAPI) are available at:
   ```
   http://127.0.0.1:8000/docs
   ```

5. Upload a currency note image (PNG/JPG) — it will scan and show **Genuine** or **Counterfeit** with a confidence score.

## Notes
- The class order (`fake`, `real`) matches what `image_dataset_from_directory` assigned during training — don't change it unless you retrain differently.
- If you get a `ModuleNotFoundError`, re-run `pip install -r requirements.txt`.
- For deployment (so you can share a live link on your resume), Hugging Face Spaces or Render both support FastAPI apps for free.
